Welcome to articull's documentation!
====================================

Overview
--------

**ArtiCull** (`Arti`\fact `Cull`\er) is a variant call refinement algorithm for single-cell
whole-genome sequencing (scWGS) data. It enables single nucleotide variant (SNV) based analyses in scWGS, 
including mutational signature and evolutionary analyses.

Citation
--------

If you use ArtiCull in your research, please cite the following manuscript:

```
Satas, G., Myers, M. A., McPherson, A., & Shah, S. P. (2025). Inferring active mutational processes in cancer using single cell sequencing and evolutionary constraints. bioRxiv. doi:10.1101/2025.02.24.639589
```
